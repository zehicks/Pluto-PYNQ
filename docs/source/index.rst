Pluto-PYNQ
==========

**Pluto-PYNQ** is a :doc:`PYNQ <pynq>` v3.0.1 embedded Linux image for the :doc:`pluto` software defined radio. The project is built using ``make``, which is documented on the :ref:`Makefile Targets` page.
